# HTML Crash Course

## Overview

This project is a personal HTML crash course that I created while learning the fundamentals of front-end web development. It serves as a quick reference guide that I can use when building future projects.

The goal of this repository is to keep examples of common HTML structures, elements, and shortcuts that are frequently used when developing websites.

## What This Project Includes

The project contains several basic HTML pages that demonstrate common website structures and components.

Files included in this repository:

* **index.html** – Main homepage structure
* **about.html** – Example about page layout
* **contact.html** – Basic contact page structure
* **assets/images** – Folder for images used in the project

These pages help demonstrate how a small multi-page website can be organized using HTML.

## Purpose

The purpose of this repository is to:

* Practice writing HTML from scratch
* Learn how to structure a multi-page website
* Keep reusable HTML snippets for future projects
* Improve familiarity with development tools and workflows

This project also helped me practice using version control with Git and publishing projects to GitHub.

## Tools Used

* HTML5
* Git
* GitHub
* Visual Studio Code

While building this crash course project, I practiced:
